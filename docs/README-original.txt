
Compilador para el lenguaje Cascabel versión 1.1 (alfa)
Facultad de ingeniería - UNAM
Lenguajes Formales y Autómatas



Requerimientos:

Python 3 [ https://www.python.org/downloads ]
NLTK (Natural Language Tool Kit) [ https://www.nltk.org/install.html ] [ pip install nltk ]
RPLY [ https://rply.readthedocs.io/en/latest/ ] [pip install rply]


Para ejecutar el compilador basta con ejecutar el archivo main.py

El compilador le solicitará el nombre del programa a compilar, el cual deberá estar en el directorio programas, si no ingresa ningún nombre de programa, el compilador tomará por default el programa holamundo.casc. 

Posteriormente el compilador solicitará al usuario el número de etapa hasta la cual se desea ejecutar el compilador (default: 1). Las opciones son:

(1) Análisis léxico
(2) Análisis sintáctico
(3) Generación semántico
(4) Generación de código
(5) Ejecución del programa objeto
